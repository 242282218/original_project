# Frontend Style Guard Workflow

来源 Skill: `.trae/skills/fe-style-guard/SKILL.md`

## 使用场景

- 涉及 CSS / Tailwind / 样式系统调整时

## 执行步骤

1. **识别项目样式方案**
   - CSS Modules
   - Styled Components
   - Tailwind CSS
   - Vanilla CSS

2. **检查样式一致性**
   - 是否混用多种写法
   - 是否遵循项目规范

3. **输出说明**
   - 使用的样式方案说明
   - 一致性检查结果

## 禁止事项

- 不引入新样式体系
- 不混用多种写法
