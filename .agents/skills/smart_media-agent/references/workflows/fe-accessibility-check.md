# Frontend Accessibility Check Workflow

来源 Skill: `.trae/skills/fe-accessibility-check/SKILL.md`

## 使用场景

- 需要满足基本可访问性（a11y）要求时

## 执行步骤

1. **检查语义结构**
   - HTML 语义标签使用
   - 标题层级结构
   - 列表和表格结构

2. **检查可访问性问题**
   - 图片 alt 属性
   - 表单标签关联
   - 焦点管理
   - 键盘导航支持

3. **输出 a11y 问题清单**
   - 问题描述
   - 问题位置
   - 修复建议

## 禁止事项

- 不直接修改代码（仅分析和报告）
