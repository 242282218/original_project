# Frontend API Contract Workflow

来源 Skill: `.trae/skills/fe-api-contract/SKILL.md`

## 使用场景

- 前端接口调用异常
- 前后端数据不一致

## 执行步骤

1. **核对接口参数**
   - 请求参数类型
   - 必填/可选参数
   - 参数命名一致性

2. **核对返回值**
   - 响应数据结构
   - 字段类型
   - 嵌套对象结构

3. **检查错误处理**
   - 错误码定义
   - 错误消息格式
   - 异常场景处理

4. **输出分析报告**
   - 使用中的接口字段
   - 潜在不一致点

## 禁止事项

- 不修改后端接口
