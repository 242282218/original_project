# Test Coverage Workflow

来源 Skill: `.trae/skills/test-coverage/SKILL.md`

## 使用场景

- 评估测试完整性
- 发现测试缺口
- CI/CD 门禁检查
- 发布前验证
- 持续改进

## 执行步骤

1. **运行覆盖率分析**
   - 配置覆盖率工具
   - 执行测试并收集数据

2. **生成覆盖率报告**
   - 行覆盖率
   - 分支覆盖率
   - 函数覆盖率

3. **识别未覆盖代码**
   - 列出未覆盖的行
   - 分类原因

4. **缺口分析**
   - 关键缺口（高风险）
   - 低优先级缺口

5. **制定改进计划**
   - 优先级排序
   - 预估测试数量
   - 预期覆盖率提升

## 最佳实践

1. **质量优于数量**: 100% 覆盖不等于无 bug
2. **关注边界**: 优先覆盖边界和异常
3. **持续监控**: 集成到 CI/CD
4. **增量检查**: 新代码必须覆盖
