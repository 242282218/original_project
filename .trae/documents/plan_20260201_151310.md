## 使用在线 pansou 服务集成计划

### 目标
配置并测试使用在线 pansou 服务：`http://pansou.xzcccc.eu.org/`

### 执行步骤

**步骤1：修改默认 API URL**
- 修改 `quark_strm/app/services/search_service.py`
- 将默认 URL 从 `http://localhost:8888` 改为 `http://pansou.xzcccc.eu.org`

**步骤2：测试搜索接口**
- 调用 pansou 健康检查：`http://pansou.xzcccc.eu.org/api/health`
- 测试搜索功能：`http://pansou.xzcccc.eu.org/api/search?kw=测试`

**步骤3：验证集成**
- 启动 smart_media 服务
- 调用 `/api/search/status` 检查服务状态
- 调用 `/api/search?keyword=测试` 验证搜索功能

### 预期结果
- ✅ 搜索服务成功连接到在线 pansou
- ✅ API 返回搜索结果
- ✅ 原有 API 接口保持兼容
