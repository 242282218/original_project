## 修复计划

### 目标
启用 Telegram 和 NetSearch 搜索源，从多个渠道获取资源，支持多种网盘类型。

### 问题分析
- `b.funletu.com` API 本身只返回夸克网盘链接
- 当前配置只启用了 quark_api 一个搜索源
- 需要启用 telegram 和 net_search 源来获取其他网盘资源

### 执行步骤

#### 步骤1: 启用多搜索源配置
**文件**: `quark_strm/app/core/sdk_config.py`
- 修改 `create_search_service` 方法
- 启用 telegram 和 net_search 搜索源
- 添加配置支持从配置文件读取搜索源设置

#### 步骤2: 检查并配置 Telegram 搜索源
**文件**: `quark_api_package/packages/search/sources/telegram.py`
- 检查 Telegram 源的配置要求
- 确保能正确提取多种网盘链接

#### 步骤3: 检查并配置 NetSearch 搜索源
**文件**: `quark_api_package/packages/search/sources/net_search.py`
- 检查网络搜索源的配置
- 确保能正确提取多种网盘链接

#### 步骤4: 添加搜索源选择功能到前端
