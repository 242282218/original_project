## 问题根源

前端筛选条件（cloudTypes）改变后不会自动触发重新搜索。

## 修复方案

在 `SearchView.vue` 中添加 `watch` 监听 `filters.cloudTypes` 变化：

1. 导入 `watch` from 'vue'
2. 添加 watch 监听 `filters.cloudTypes`
3. 当已经搜索过（hasSearched=true）时自动重新搜索

## 修改内容

```javascript
// 1. 导入 watch
import { ref, reactive, watch } from 'vue'

// 2. 添加 watch（在 setup 中）
watch(
  () => filters.cloudTypes,
  () => {
    if (hasSearched.value) {
      handleSearch()
    }
  },
  { deep: true }
)
```
