# Test Failure Workflow

来源 Skill: `.trae/skills/test-failure/SKILL.md`

## 使用场景

- 测试失败
- CI 不稳定
- 出现 flaky test

## 执行步骤

1. **收集失败信息**
   - 失败测试列表
   - 错误信息
   - 堆栈追踪

2. **分析失败原因**
   - 代码问题
   - 环境问题
   - 时序/并发问题
   - 依赖问题

3. **识别不稳定测试**
   - 随机失败的测试
   - 时间敏感的测试

4. **输出分析报告**
   - 失败根因分析
   - 不稳定测试列表
   - 影响范围说明

## 禁止事项

- 不直接修代码
- 不删除测试
