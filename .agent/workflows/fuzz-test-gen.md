# Fuzz Test Generator Workflow

来源 Skill: `.trae/skills/fuzz-test-gen/SKILL.md`

## 使用场景

- 安全敏感代码测试
- 解析器/解码器测试
- 网络协议实现
- 文件格式处理
- 发现内存安全问题

## 执行步骤

1. **选择模糊测试工具**
   - LibFuzzer
   - AFL
   - Honggfuzz

2. **创建测试 harness**
   - 编写 fuzz 入口函数
   - 处理输入数据

3. **准备语料库**
   - 收集有效输入样本
   - 覆盖不同格式

4. **执行模糊测试**
   - 运行指定时长
   - 监控崩溃

5. **分析崩溃报告**
   - 分类崩溃类型
   - 定位问题代码
   - 生成修复建议

## 最佳实践

1. **语料库质量**: 提供多样化的初始输入
2. **持续运行**: 模糊测试应该持续运行
3. **崩溃分析**: 每个崩溃都要分析修复
4. **回归测试**: 崩溃样本加入回归测试
