# Frontend Read Only Workflow

来源 Skill: `.trae/skills/fe-read-only/SKILL.md`

## 使用场景

- 需要理解前端项目整体结构
- 了解技术栈、页面与组件关系
- 不希望 AI 对代码产生任何修改时

## 执行步骤

1. **分析技术栈**
   - 框架：React / Vue / Next / Vite 等
   - 状态管理方案
   - 样式方案

2. **分析页面与路由结构**
   - 路由配置
   - 页面组织方式

3. **分析组件组织方式**
   - 组件目录结构
   - 共用组件库

4. **分析构建与启动方式**
   - package.json scripts
   - 构建配置

5. **输出分析报告**
   - 技术栈概览
   - 页面/路由结构
   - 主要组件职责

## 禁止事项

- 不修改代码
- 不提出重构或优化建议
- 不执行命令
