# Frontend Safe Refactor Workflow

来源 Skill: `.trae/skills/fe-safe-refactor/SKILL.md`

## 使用场景

- 需要优化前端代码结构
- 必须保证 UI 与行为完全一致时

## 执行步骤

1. **确定重构范围**
   - 目标组件/模块
   - 影响的文件

2. **执行安全重构**
   允许的操作：
   - 拆分组件
   - 抽取 hooks
   - 重命名（语义等价）

3. **验证行为一致**
   - 对比重构前后的渲染结果
   - 确认交互行为一致

4. **输出报告**
   - 修改文件列表
   - 行为不变说明

## 禁止事项

- 不改变 UI
- 不改变交互
- 不引入新设计
