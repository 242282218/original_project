# Frontend Test Focus Workflow

来源 Skill: `.trae/skills/fe-test-focus/SKILL.md`

## 使用场景

- 修复前端行为或 UI bug 前

## 执行步骤

1. **分析现有测试**
   - 查找相关测试文件
   - 检查测试覆盖情况

2. **评估测试需求**
   - 是否需要补充测试
   - 测试的优先级

3. **补充或调整测试**
   - 优先分析或补充测试
   - 确保测试覆盖问题场景

4. **输出报告**
   - 相关测试说明
   - 测试覆盖情况

## 禁止事项

- 不写测试就改核心逻辑
